React与jQuery的思想对比2

2020-04-20

jQuery with React

React的组件化思想要明显强于jquery，虽然在jquery中也可以通过函数的方式去创造组件，但是却无法使得组件能够自动刷新，想要刷新只能去
显示的执行（先移除dom，在创建dom，在页面重绘上代码比较大）。而React则无需去手动刷新，会根据传入的字段或者状态的改变去自动刷新，提高了
开发速度，也提高了页面刷新的速度。
React很清晰的将数据层与视图层分离开来
